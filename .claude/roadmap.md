# ROADMAP

## Sprint 1

Goal:
Users can create projects and tasks.

Features:

* Authentication
* Workspace
* Project CRUD
* Task CRUD
* Kanban Board

Success Criteria:

A user can:

1. Sign in
2. Create a workspace
3. Create a project
4. Create tasks
5. Drag tasks between columns
