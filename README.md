# Aspen OS

The simplest and most enjoyable project operating system for nonprofit and community organizations.

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS · shadcn/ui · Supabase · Vercel.

## Getting Started

```bash
npm install
cp .env.example .env.local   # fill in your Supabase project URL + anon key
npm run dev
```

Open http://localhost:3000.

See [`docs/`](docs) for the full Sprint 1 planning set, including the build order and execution plan currently driving development.
